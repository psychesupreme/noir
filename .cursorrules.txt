Project: Noir & Bloom (Kenyan Floral & Gifting ERP)
    Stack: Laravel 11, Livewire, Tailwind CSS, MySQL

    Rules:
    1. Currency is Kenyan Shillings (Ksh).
    2. Timestamps must align with EAT (East Africa Time).
    3. Phone validations must accept Kenyan formats (e.g., 2547..., 07...).
    4. Write clean, strongly-typed PHP following the Single Responsibility Principle.
    5. Keep UI minimalist (Dark/Light mode support).
    6. Do not over-engineer database columns unless specified.
